h1. JsonBenchmarkCpp 

JsonBenchmarkCpp is a small program to benchmark various JSON
frameworks in c++

Currently supports following libs,

# 1. [uxs::db::json](https://github.com/gbuzykin/uxs) (2e0aaa9004 Fri Dec 26 17:24:45 2025 +0300)
# 2. [rapidjson](https://github.com/Tencent/rapidjson) (24b5e7a8b2 Sun Dec 22 15:53:24 2024 +0200)
# 3. [jsoncpp](https://github.com/open-source-parsers/jsoncpp) (b511d9e649 Wed Nov 12 09:14:04 2025 +0100)
# 4. [nlohmann](https://github.com/nlohmann/json) (55f93686c0 Fri Apr 11 10:42:28 2025 +0200)
# 5. [libjson](https://github.com/mousebird/libjson) (bdb42ade3b Tue Mar 6 13:35:54 2018 -0800)
# 6. [cajun](https://github.com/cajun-jsonapi/cajun-jsonapi) 8a26ad3a17 Wed Jun 16 22:40:16 2021 +0200)
# 7. [json_spirit](https://github.com/png85/json_spirit) (5e16cca59b Sun Sep 6 00:14:53 2015 +0200)
# 8. [json-parser](https://github.com/udp/json-parser) (8ac4477ad3 Fri May 9 20:01:35 2025 -0500)
# 9. [avery](https://github.com/matthewaveryusa/averyws)

See my "blog post":http://www.lijoantony.com/?p=5 for details.

h2. Building

bc. $ git clone git://github.com/gbuzykin/JsonBenchmarkCpp.git
$ cmake -S . -B build
$ cmake --build build
$ cmake --install build --prefix install 

h3. Running

bc. $ cd testdata
$ ../install/bin/JsonBenchmarkCpp
